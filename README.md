# EmlakProjesi

## Emlak Otomasyonu projesinde genel olarak
* Giriş Ekranı 
* Login Ekranı

* İlan arama, ilan ekleme, ilan silme, talep bildirme, eşleşen talepleri görüntüleme gibi işlemler gerçekleştirilmiştir.
